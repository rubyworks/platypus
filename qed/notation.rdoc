= Overloadable

First we need to load the library.

  require 'overloadable'

Now we can define a class that utilizes it.

  class X
    include Overloadable

    sig String, String

    def x(str1, str2)
      str1.assert.is_a?(String)
      str2.assert.is_a?(String)
    end

    sig Integer, Integer

    def x(int1, int2)
      int1.assert.is_a?(Integer)
      int2.assert.is_a?(Integer)
    end
  end

Now lets see if it worked.

  x = X.new

  x.x("Hello", "World")

  x.x(100, 200)
