DESCRIPTION

Provides a generic simple type conversion utility. All the ruby core conversions are available by default.

FEATURES/ISSUES

  • Uses doulble-dispatch.
  • Fully generic type conversion system.
  • Built-in support for most core classes.

RELEASE NOTES

Please see HISTORY file.

SYNOPSIS

   "1234".cast_to Float     => 1234.0  (Float)
   Time.cast_from("6:30")   => 1234.0   (Time)

INSTALLATION

To install with RubyGems simply open a console and type: 

  gem install typecast

Local installation requires Setup.rb (gem install setup), then download the tarball package and type: 

  tar -xvzf typecast-1.0.0.tgz
  cd typecast-1.0.0
  sudo setup.rb all

Windows users use ‘ruby setup.rb all’.
